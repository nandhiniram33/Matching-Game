# Animal Kingdom Memory

Premium jungle-themed memory matching game built with React, Tailwind, Framer Motion and GSAP (optional).

## Quick start

1. Install dependencies

```bash
npm install
```

2. Run dev server

```bash
npm run dev
```

## Notes
- Tailwind and Vite are configured.
- Add your Firebase config to `src/firebase.js`.
- Add audio assets to `public/assets` and reference them in components.
- This scaffold includes core game logic and UI components to extend.

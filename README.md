# 🐾 Matching Game

An interactive **animal-themed memory card game** built with **React, Vite, Tailwind CSS, and Firebase**. Players challenge their memory by matching animal card pairs across multiple difficulty levels while earning achievements, tracking statistics, unlocking collections, and competing on a global leaderboard.

## ✨ Features

* 🎮 Interactive memory card matching gameplay
* 🐻 Animal-themed cards with engaging visuals
* 🏆 Multiple difficulty levels
* ⏱️ Timer-based scoring system
* 🥇 Global leaderboard
* 🎖️ Achievement and rewards system
* 📊 Player statistics and progress tracking
* 🦁 Animal collection and unlockables
* ⚙️ Customizable game settings
* 🔐 Firebase Authentication (Login & Registration)
* 🎵 Sound effects, background music, and particle animations
* 📱 Fully responsive design for desktop and mobile

## 🛠️ Tech Stack

* **Frontend:** React, Vite
* **Styling:** Tailwind CSS, PostCSS
* **Backend:** Firebase Authentication & Firestore
* **Language:** JavaScript (ES6+)

## 🚀 Getting Started

```bash
git clone <repository-url>
cd matching-game
npm install
npm run dev
```

The application will start on:

```
http://localhost:5173
```

## 🎯 Gameplay

Flip two cards at a time to find matching animal pairs. Complete each level in the fewest moves and shortest time to earn higher scores, unlock achievements, and climb the leaderboard.
